# Retrieve a list of shipping methods with query builder

### URL

```text
api/v1/shipping-methods
```

### Method

```text
GET
```





### Auth required

```text
Yes
```

### Headers

```text
Content-Type: application/json
Accept: application/json
Authorization: Bearer ******************

```
