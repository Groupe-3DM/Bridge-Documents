# Retrieve a list of item status with query builder

### URL

```text
api/v1/item-statuses
```

### Method

```text
GET
```





### Auth required

```text
Yes
```

### Headers

```text
Content-Type: application/json
Accept: application/json
Authorization: Bearer ******************

```

| item_status_id | name     |
|----------------|----------|
| 1              | Active   |
| 2              | Inactive |
| 3              | Archived |
