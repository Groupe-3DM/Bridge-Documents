# Retrieve a list of quantity types with query builder

### URL

```text
api/v1/quantity-types
```

### Method

```text
GET
```

### Auth required

```text
Yes
```

### Headers

```text
Content-Type: application/json
Accept: application/json
Authorization: Bearer ******************

```
